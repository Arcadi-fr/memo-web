JavaScript

Javascript est un langage de script mutliplateforme et orienté objet. C'est un langage léger qui doit faire partie d'un environnement hôte (navigateur web par exmpel) pour qu'il puisse être utilisé sur les objets de cet environnement.

    Création du fichier

  • Le fichier JS doit toujours être lu en dernier. On le met à la fin du body. Si le JS est placé dans le head il faut rajouter un defer pour demander une lecture diféré : <script defer src="chemind'acces"> </scipt>
  • Le JS ne supporte pas la casse. Pour une variable, utiliser le kamelCase.

    Variable

  • Une variable est une "boîte", un espace, dans laquelle on stocke des éléments. En donnant un nom à cette boîte on peut l'appeler.
  • Pour initaliser la variable, on utilise plus var
    Ancienne syntaxe : var unTexte = "Voici un texte"
  • La nouvelle syntaxe inclus variable et constante.
    Constante : const prenom = "Justine"; Variable : let unChiffre = "24";
  • Concaténation pour ajouter une variable à une autre variable : let nouvelleChaine = "Chaine précédente : " + chaine;

    Les types de données en JS

  • Chaine de caractère : let chaineDeCaractere = "Je suis une chaine 1234567890";
  • Chiffres : let number = 24;
  • Booléen : let boolean = true;
  • Array : let array = ["je", "suis", 24, false ];
  • Object : let object = {
    prenom: "Amaury",
    age: 20,
    ville: "Brest",
    }
  • Null : let arbre = null;
  • Pour connaître le type de variable : console.log(typeof array);
