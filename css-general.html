CSS

Création du fichier CSS

    Lier le css au html

  • <link rel="stylesheet" href="./style.css"> ou
  • <style type="text/css">
    @import url(styles/habillage.css);
    @import url(styles/texte.css);
    </style>

    Bases du fichier CSS

  • Déclarer des variables :
    :root {
    --color-1: couleur }
    On les appellent par : attribut: var(--color-1)
  • Déclarer les fontes :
    @font-face {
    font-family: "lato-thin";
    src: url(./fonts/Lato-Thin.ttf);
    }
  • Annuler les éffets du user-agent (Attention aux éffets sur les bouttons):
    margin: 0;
    padding: 0;
    autres si nécessaires
  • Pour que le flex fonctionne bien :
    box-sizing: border-box
    Plus d'informations sur MDN

    • D'autres élements généraux selon le type de page et la récurrence des moficiations à faire. Par exemple s'il y a beaucoup de li on peut déclarer généralement le type de puce. Après les déclarations générales "*", on peut donner les généralités pour les ancres, les bouttons les titres, les paragraphes etc.

Exemple "*"

* {
margin: 0;
padding: 0 ;
box-sizing: border-box;
font-family: arial;
list-style-type: \261E;
}

Exemple spécifique

button {
background: none;
outline: none;
border: 1px solid white;
color: white;
text-transform: uppercase;
font-family: "roboto-regular";
transition: 0.4 ease;
cursor: pointer;
}

button:hover {
color: black;
background: white;
}

a {
text-decoration: none;
}

body {
margin: 0 auto;
text-align: center;
}

p {
font-size: 0.9rem;
}

Insérer une image en background

background: url(./img/banner.jpg) center/cover; On peut aussi attribuer "right/cover" par exmple et ajouter "fixed avant l'emplacement".

Pour appeler un élément

  • Par la classe : .nomdelaclasse
  • Par l'ID : #nomid
  • Par sa balise html : p

Pour affiner

  • On peut préciser le type de la balise html : input[type="text"]
  • on peut ne viser que les enfants et pas les petis-enfants : section > .enfantsseulement
  • On peut viser plusieurs élements : section, #unid, .uneclasse
  • On peut etre très précis : nav ul .apropos
