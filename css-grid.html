Grids

    Pour déclarer la grille

  • display: grid sur le parent de la grille.
  • grid-template-columns: On peut déclarer des fractions ( 1fr, des pourcentages ou des pixels ).br On peut aussi répéter pour ajouter une ou des colonnes : repeat(2, 1fr) içi, on ajoute 2 fois une fraction.
  • grid-template-rows: idem aux columns.
  • grid-gap: 10px Pour créer un espacement entre les items.
  • grid-column: Pour imposer un ordre dans la grille.

Déclaration sur les enfants.

grid-column: Pour imposer un ordre dans la grille.
grid-column: 1 / 4 Permet de faire un "rowspan".
grid-column: 2 / span 2 Permet encore un "rowspan".

Grid-template-area

Outils puissant pour dessiner des grilles. Dans un premier temps on dessine la grille avec grid-template-columns puis on délcare le noms des enfants sur la grille area avec grid-area.
