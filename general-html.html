  • Ajouter des row : <tr>
  • Dans lesquelles on ajoute des cellules : <td>

Les formulaires

  • Appeler avec : <form action="" method="">
  • Placer un input : <input id="" type="" placeholder="">
    Le placeholder affiche le texte par défaut.
  • Placer un label - facultatif : <label for="">
    Il faut associer le label à son input en rensignant l'id dans le for.
  • On peut paramètrer l'input avec un minimum de valeur, des step etc :
    <input id="age" type="number" min="0" max="99" step="1" value="18">
  • On peut paramétrer un input de type "select" pour faire un menu déroulant :
    <select id="gender">
    <option value="homme">Homme</option> <option selected value="femme">Femme</option>     Le "selected" donne la valeur par défaut.
  • On peut paramètrer un iput de type radio :
    <input type="radio" name="type" id="human">
    <label for="human">Humain</label>
    <input type="radio" name="type" id="dog">
    <label for="human">Chien</label>
    Lier les inputs radio avec un name est important si l'on veut qu'ils complémentaire à la sélection. Sans cela, les deux sont sélectionnable.
  • On peut placer un input de type checkbox : <input type="checkbox"> On peut intégrer la checkbox entre les deux baises input.
  • Placer la textarea : <textarea cols="30" rows="10" placeholder="Entrez votre texte">
  • Placer le submit : <input type="submit" value="valider">
