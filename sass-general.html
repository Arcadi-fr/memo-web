SASS

SASS est un préprocesseur de CSS. On code en SASS et le code sera transformé en CSS pour être intérprété par le navigateur. Cela permet de coder plus vite du CSS de par l'architecture de SASS. Il faut installer un compiler. Sur VS, "Live sass compiler".

Responsive en SASS

Etre responsive en SASS, il faut inclure la média query à l'interieur de la balise :
h1 {
text-align: center;
@media screen and (max-width: 700px) {br font-size: 2rem;
}
}

Déclarer des variables

Il suffit de donner le nom de la variable et la caractériser. Le nom est précédé de "$". :
$color1: pink;

Mixin

C'est une variable dans,laquelle on peut mettre plusieurs lignes.
@mixin center {
display: flex;
justify-content: center;
align-items: center;
} Pour appeler la mixin, on utilise "@include". Attention à ne pas mettre ":"
section {
height: 600px;
@include: center;
}
